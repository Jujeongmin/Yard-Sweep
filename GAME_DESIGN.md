# Yard Sweep 개정 기획서

초기 기획서를 기준으로 실제 개발 과정에서 확정되거나 변경된 내용을 기록한다. 별도로 적지 않은 시스템은 초기 기획을 유지한다.

## 현재 핵심 플레이 루프

1. 지역의 일반·희귀 오브젝트를 청소한다.
2. 코인과 보석을 획득한다.
3. 상점에서 장비와 플레이어 능력을 강화한다.
4. 진행도 100% 달성 시 다음 지역을 해금한다.
5. 3지역까지 완료한 뒤 1지역부터 반복 플레이한다.

## 장비 및 인벤토리 변경

- 인벤토리는 6칸에서 8칸으로 확장되었다.
- PC 장비 단축키는 숫자키 1~8이다.
- 1~6번은 일반 장비다.
- 7번 `네온 낫`과 8번 `네온 곡괭이`는 외형 스킨이 아닌 독립 프리미엄 장비다.
- 네온 낫은 잔디 전용이며 일반 낫보다 넓고 빠르다.
- 네온 곡괭이는 돌 전용이며 이동하면서 사용할 수 있다.
- 네온 낫 가격은 보석 120개, 네온 곡괭이 가격은 보석 150개다.
- 프로토타입 테스트 중에는 1~6번 일반 장비가 자동 해금된다.
- 프로토타입 테스트 중에는 저장 데이터와 무관하게 지역 1~3이 모두 해금된 상태로 시작한다.
- 신규 저장 데이터에는 테스트용 보석 300개가 지급된다. 정식 경제 밸런싱 시 제거하거나 변경한다.

## 지역 진행 변경

- 새 지역은 코인 구매가 아니라 이전 지역 진행도 100% 달성으로 자동 해금된다.
- 지역 1 완료 시 지역 2, 지역 2 완료 시 지역 3이 열린다.
- 지역 3 완료 후 앞 마당부터 다시 플레이할 수 있다.
- 상점의 `지역` 탭에서 해금된 지역을 자유롭게 이동할 수 있다.
- 지역별 남은 오브젝트 수량과 진행률은 이동·새로고침 후에도 유지된다.
- 완료 보상을 받은 지역의 진행 데이터만 삭제되며, 재입장 시 새 오브젝트가 생성된다.
- 지역별 일반 오브젝트 수량은 초기 기획을 유지한다.
- 지역 완료 시 전용 UI와 효과음이 재생된다.
- 진행도(%) 표시는 반올림이 아닌 내림(`Math.floor`)을 사용한다. 오브젝트 총량이 큰 지역(예: 돌 정원)에서 반올림을 쓰면 마지막 1개가 남아도 화면에 100%로 보여 완료 카드가 안 뜨는 것처럼 보이는 문제가 있었다.
- 상단 HUD에 현재 지역 진입 후 경과 시간(mm:ss)을 표시한다.

## 지역별 비주얼 테마

세 지역이 같은 플레이 공간(지형·집·나무)을 공유하지만 지역 진입 시 하늘·안개·바닥·산책로 색상과 장식 오브젝트를 바꿔 지역마다 다른 분위기를 낸다.

- **앞 마당**: 기존 파란 하늘 + 초록 잔디 + 황토색 산책로.
- **정원**: 청록빛 하늘, 짙은 초록 잔디, 흩뿌려진 꽃 장식(색색의 꽃 36개)과 산책로 양옆 산울타리.
- **돌 정원**: 흙빛 하늘·안개, 회갈색 바닥·산책로, 바닥에 놓인 납작한 디딤돌 장식(18개)으로 마른 정원(zen garden) 느낌을 낸다. 처음엔 청소 가능한 `돌` 오브젝트와 똑같이 생긴 둥근 바위를 썼다가 서로 헷갈린다는 피드백을 받아 납작한 판석으로 바꿨고, 이후 우뚝 선 뾰족한 입석 바위도 추가했다가 다시 제거해 지금은 납작한 디딤돌만 남아있다.
- 장식은 미리 만들어 두고 해당 지역이 아닐 때는 `visible = false`로 숨긴다(청소 가능한 오브젝트가 아닌 순수 배경 장식이다).

## 지역 반복 완료 보상

지역을 완료할 때마다(반복 플레이 포함) 코인·보석 보너스를 지급한다. 다음 지역 해금과 별개로 매번 지급된다.

| 지역 | 코인 | 보석 |
|---|---|---|
| 앞 마당 | 30 | - |
| 정원 | 60 | 2 |
| 돌 정원 | 120 | 5 |

- 코인 획득량 업그레이드는 완료 보상 코인에도 적용된다. 보석에는 적용되지 않는다.

## 코인/보석 표시 방식

- 코인·보석은 내부적으로 소수점을 유지한 채 누적된다(업그레이드 배율 등으로 생기는 소수점을 매번 반올림하면 장기적으로 손해가 누적되기 때문). 화면에는 `Math.floor`로 정수만 표시한다.
- 업적용 누적 스탯(`stats.coinsEarned`)은 표시가 정수여야 하므로 정수로 누적한다.

## 청소 중단/재개 규칙

- 조준(에임)에서 오브젝트가 벗어나거나 청소를 멈추면 진행 중이던 청소는 0%로 초기화된다.
- 다만 화면 경계에서 순간적으로 깜빡이는 것까지 중단으로 처리하면 작업음이 반복 재생되는 문제가 있어, 0.15초의 유예 시간(grace period)을 두고 그 안에 다시 잡히면 중단으로 치지 않는다.

## 미션과 업적

- 상점에 `미션`, `업적` 탭을 추가했다.
- **미션(단기 목표)** : 낙엽 100개 청소, 빈 캔 30개 수거, 마당 100% 청소, 5분 안에 지역 완료. 초기 기획의 "3개 무작위 로테이션" 대신, 4개 미션 풀 전체를 항상 동시에 노출하는 방식으로 단순화했다.
- **업적(장기 목표)** : 첫 청소 완료, 코인 1,000개 획득, 모든 지역 해금, 낙엽 10,000개 제거. 누적 스탯에서 진행도를 계산한다.
- 목표를 달성하면 버튼을 누를 필요 없이 그 즉시 자동으로 보상이 지급되고 화면에 알림 팝업이 뜬다(`checkMissionsAndAchievements()`가 오브젝트 청소·지역 완료 시점마다 모든 미션/업적 진행도를 검사). 미션은 보상 지급 후 진행도가 0으로 초기화되어 다시 반복되고, 업적은 한 번 지급되면 다시 초기화되지 않는다. 상점의 미션/업적 탭 버튼은 이제 클릭 가능한 "받기" 버튼이 아니라 진행도/완료 상태만 보여주는 표시용이다.
- 미션·업적 보상에도 코인 획득량 업그레이드가 코인 보상에 적용된다(보석 제외).

## 레벨 시스템 (뼈대)

- 오브젝트를 하나 청소할 때마다 경험치 1%가 오르며, 100%마다 레벨이 1 오른다. 오브젝트 종류에 따른 차등은 없다.
- 레벨업 보상은 없다. 순수 진행도 표시/경쟁용 지표다.
- 별도 저장 필드 없이 누적 청소 개수(`stats.totalCleaned`)에서 파생 계산한다: `레벨 = floor(누적 청소 개수 / 100) + 1`, `진행도% = 누적 청소 개수 % 100`.
- 상단 HUD에 레벨과 경험치 바를 항상 표시한다.
- 상점에 `랭킹` 탭을 추가했다. 현재는 본인 레벨만 표시하며, 실제 글로벌 랭킹은 배포 플랫폼(Verse8)의 랭킹 기능과 연동 예정이다.
- `reportScoreToPlatform()` 함수가 레벨이 바뀔 때마다 호출되도록 뼈대만 만들어 두었다. Verse8 랭킹 API 연동 방법이 확인되면 이 함수 안에 실제 호출을 채워 넣는다.

## 희귀 오브젝트

각 대상은 조건을 만족하는 지역에서 종류별 1~2개가 무작위 생성되며 지역 진행도에 포함된다.

| 오브젝트 | 등장 지역 | 사용 장비 | 보상 |
|---|---|---|---|
| 금 캔 | 모든 지역 | 빗자루·청소기 계열 | 코인 15 |
| 골드 상자 | 모든 지역 | 모든 장비 | 코인 40 |
| 보석 상자 | 모든 지역 | 모든 장비 | 보석 10 |
| 황금 돌 | 돌이 등장하는 지역 3 | 곡괭이 계열 | 코인 50 + 보석 5 |

- 코인 획득량 업그레이드는 희귀 오브젝트의 코인 보상에도 적용된다.
- 보석 보상에는 코인 획득량 업그레이드가 적용되지 않는다.

## 프리미엄 재화 아이템

- **코인 부스터** (보석 30개): 구매 시점부터 30분간 코인 획득량이 100% 추가된다(청소·지역 완료·미션·업적 보상 전부 포함, 코인 획득량 업그레이드와 중첩 적용). 상단 HUD에 남은 시간이 표시된다. 활성 중 다시 구매하면 남은 시간에 30분이 더해진다.
- **로봇청소기** (보석 200개): 장비 슬롯을 차지하지 않는 일회성 구매형 보유 아이템. 보유하면 게임 플레이 중 백그라운드에서 항상 작동하는 자율 AI 유닛으로, 플레이어와 무관하게 자기 자신의 월드 좌표를 기준으로 지역 전체에서 가장 가까운 미청소 오브젝트를 찾아 그쪽으로 스스로 이동한 뒤(2.2유닛/초), 도착하면 5초에 하나씩 청소하고 코인·보석을 지급한다. 장비 종류 제한 없이 모든 오브젝트를 청소할 수 있다. 밸런스를 위해 의도적으로 느리게(5초/개) 설계했다.
- 플레이어를 따라다니지 않는다 — 처음엔 카메라를 따라다니는 방식으로 만들었다가(1인칭이라 화면에 아예 안 보이는 문제도 있었음), 실제 자율 청소 로봇처럼 보이도록 독립적인 위치·이동 로직으로 다시 만들었다. 장애물 회피/경로 탐색 없이 목표 오브젝트를 향해 직선으로만 이동한다.
- 3D 모델(GLB)을 로드해 표시하며, 지역을 이동하면 이전 지역의 목표(오브젝트 참조)는 초기화되고 새 지역에서 다시 탐색한다.
- 실제로 오브젝트를 청소하는 동안(이동 중에는 재생하지 않음) 송풍기 작업음을 재생한다. 플레이어가 직접 송풍기를 사용할 때보다 조용하도록 기본 볼륨의 0.5배로 재생한다.

## 사운드 시스템

- 지역 1~3에 각각 별도의 BGM을 사용한다.
- 빗자루, 진공청소기, 낫, 곡괭이 계열은 서로 다른 반복 작업음을 사용한다.
- 모든 UI 버튼에 버튼음을 사용한다.
- 지역 완료 시 전용 완료음을 사용한다.
- 실제 청소 대상이 범위 안에 있을 때만 작업음이 재생된다.
- 코인·보석을 획득하는 모든 순간(청소, 지역 완료, 미션·업적 수령)에 코인 획득음이 재생된다.
- 이동 중일 때만 발걸음 소리가 루프 재생되고, 멈추면 즉시 정지한다.
- 점프가 삭제되어 점프음은 없으며, 착지음(점프·착지음 파일)도 함께 제거되었다.

## 이동/조작 변경

- 점프 기능을 완전히 삭제했다(PC `Space` 키, 모바일 점프 버튼, 관련 중력·착지 로직 모두 제거). 플레이어는 항상 지면 높이에 고정된다.

## UI 변경

- 좌측 상단에 코인과 보석 잔액을 항상 표시한다.
- 코인 부스터가 활성화되어 있으면 코인·보석 옆에 남은 시간 배지가 표시된다.
- 상점은 `장비`, `플레이어 강화`, `지역`, `미션`, `업적`, `랭킹`, `프리미엄` 탭으로 구성된다.
- 상단 인벤토리는 일반 장비 6개와 프리미엄 장비 2개를 표시한다.
- 낙엽 모양을 4개 꼭짓점의 단순한 별 모양에서 5개 꼭잎(위 1개, 좌우 상단·하단 각 1개)과 줄기가 있는 단풍잎 실루엣으로 다시 그렸다.

## 설정

- 상단 `⚙` 버튼 또는 `T` 키로 설정 패널을 열고 닫는다. 상점(Tab)과 상호 배타적으로 열리며(하나를 열면 다른 하나는 자동으로 닫힘), 여는 동안 포인터 락이 해제되고 청소가 중단된다.
- **BGM/효과음 볼륨**: 각각 0~100% 슬라이더. 기존에 하드코딩되어 있던 사운드별 기본 볼륨(예: BGM 0.28, 버튼음 0.45, 청소 작업음 0.6 등)에 곱해지는 배율로 적용해, 기존 사운드 간 상대적인 믹스 비율은 그대로 유지된다. 슬라이더를 움직이면 즉시 반영된다.
- **마우스 감도**: 50~200% 슬라이더(기본 100%). PC 마우스 이동과 모바일 터치 드래그의 시점 회전 속도에 곱해지는 배율이다.
- **진동** (모바일 전용 항목, PC에서는 숨김): 켜져 있으면 오브젝트를 하나 청소 완료할 때마다 짧게 진동한다(`navigator.vibrate`).
- **언어**: 한국어/English 버튼으로 전환하면 게임 내 모든 텍스트(인벤토리, 상점 전 탭, 미션·업적, 알림, 지역 완료 카드, 힌트, 설정 패널 자체 등)가 즉시 해당 언어로 바뀐다. `src/i18n.ts`에 ko/en 전체 문자열 딕셔너리와 `t(key, vars)` 템플릿 함수가 있고, `gameData.ts`의 도구/오브젝트/지역/미션/업적 `name`·`label` 필드는 실제 표시 문자열이 아니라 i18n 키를 담고 있어 `t()`로 해석해야 한다. 정적 마크업은 `data-i18n`/`data-i18n-aria` 속성으로 표시하고, `applyLocale()`이 페이지 로드 시 한 번과 언어 버튼 클릭 시마다 전체를 다시 그린다.
- 모든 설정값은 기존 저장 데이터(`localStorage`)의 같은 블롭 안에 `settings` 필드로 함께 저장되어 새로고침 후에도 유지된다.

## 저장되는 데이터

- 코인과 보석 (소수점 포함)
- 현재 지역과 해금된 최고 지역
- 해금 장비
- 플레이어 업그레이드 레벨
- 프리미엄 장비 구매 상태
- 코인 부스터 만료 시각, 로봇청소기 보유 여부
- 누적 스탯(청소 개수 등), 미션 진행도, 업적 수령 여부
- 설정(언어 선택값, BGM/효과음 볼륨, 마우스 감도, 진동 여부)

## 아직 추가 구현이 필요한 초기 기획

- 일반 장비 외형 스킨 시스템
- 광고 보상 (가짜 광고 시뮬레이션으로 구현 예정). 청소 부스터는 "코인 부스터"로 대체 구현했다.
